\doxysubsection{Detailed Description}
Concrete observer class for staff responsible for pruning plants. 

This class represents greenhouse staff members who are specifically responsible for pruning plants. It implements the \mbox{\hyperlink{classGreenhouseStaff}{Greenhouse\+Staff}} interface and responds to notifications when plants require pruning care (care\+Type == \char`\"{}pruning\char`\"{}).
