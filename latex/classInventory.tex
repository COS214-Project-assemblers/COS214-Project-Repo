\doxysubsection{Detailed Description}
Stores a collections of plants, will return vectors of plants for the manager to choose from and then offer the client. 

The \mbox{\hyperlink{classInventory}{Inventory}} class stores all available \mbox{\hyperlink{classPlant}{Plant}} objects that are availible for sale to clients, it allows for filtering based on a categoy or by a price range.
