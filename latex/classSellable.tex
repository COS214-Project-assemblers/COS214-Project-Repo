\doxysubsection{Detailed Description}
Concrete state class representing a plant that is ready for sale. 

This class represents the state where a plant has matured sufficiently and meets all quality criteria to be sold to customers. In this state, the plant can be moved to the \mbox{\hyperlink{classSalesFloor}{Sales\+Floor}}.
