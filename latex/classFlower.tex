\doxysubsection{Detailed Description}
Represents a \mbox{\hyperlink{classFlower}{Flower}} plant type within the greenhouse system. 

The \mbox{\hyperlink{classFlower}{Flower}} class extends the abstract \mbox{\hyperlink{classPlant}{Plant}} class and provides a concrete implementation for creating and cloning \mbox{\hyperlink{classFlower}{Flower}} objects.
