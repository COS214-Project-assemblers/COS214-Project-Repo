\doxysubsection{Detailed Description}
Abstract base class defining the interface and shared behavior for all customer types. 

This class serves as the base for all specific customer types (Ignorant, Average, Green\+Finger). It maintains customer dialogue, preferences, and plant offerings, and provides functionality to serialize the customer data to JSON format for \mbox{\hyperlink{classAPI}{API}} communication.

The class provides the data structure that gets built by the Builder pattern.
