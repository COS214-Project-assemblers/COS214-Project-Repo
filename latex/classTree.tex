\doxysubsection{Detailed Description}
Represents a \mbox{\hyperlink{classTree}{Tree}} plant type within the greenhouse system. 

The \mbox{\hyperlink{classTree}{Tree}} class extends the abstract \mbox{\hyperlink{classPlant}{Plant}} base class, providing a concrete implementation for creating and cloning \mbox{\hyperlink{classTree}{Tree}} objects.
