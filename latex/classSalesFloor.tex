\doxysubsection{Detailed Description}
Manages \mbox{\hyperlink{classCustomer}{Customer}} flow and provides access to the inventory for the \mbox{\hyperlink{classManager}{Manager}} during sales interactions. 

The \mbox{\hyperlink{classSalesFloor}{Sales\+Floor}} class is for the enviroment in which the \mbox{\hyperlink{classCustomer}{Customer}} can interact with the \mbox{\hyperlink{classManager}{Manager}}. It maintains a current \char`\"{}queue\char`\"{} and history lists.
