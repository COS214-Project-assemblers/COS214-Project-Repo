\doxysubsection{Detailed Description}
Abstract base class representing different states of a plant. 

This class defines the interface for all concrete plant states in the State pattern. Each state represents a specific phase of a plant\textquotesingle{}s lifecycle (\mbox{\hyperlink{classSellable}{Sellable}}, \mbox{\hyperlink{classNotSellable}{Not\+Sellable}}) and defines behaviour specific to that state.
