Detailed Description

Concrete builder for constructing GreenFingerCustomer objects with random dialogue selection.

Implements the CustomerBuilder interface to create GreenFingerCustomer instances with randomly selected dialogues from the JSON configuration. Uses the Builder pattern to separate customer construction from customer representation.
