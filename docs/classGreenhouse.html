Detailed Description

Central greenhouse management system coordinating plants, staff, and sales.

The Greenhouse class serves as the main hub for plant management, maintaining inventory, coordinating staff activities, and interfacing with the sales floor. It follows the Facade pattern to provide a simplified interface to the complex greenhouse subsystem.
