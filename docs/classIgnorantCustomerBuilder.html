Detailed Description

Concrete builder for constructing IgnorantCustomer objects with random dialogue selection.

Implements the CustomerBuilder interface to create IgnorantCustomer instances with randomly selected dialogues from the JSON configuration. Uses the Builder pattern to separate customer construction from customer representation.
