Detailed Description

Abstract base class for greenhouse staff observers.

This class defines the interface for greenhouse staff members who need to be notified when plants require care. It follows the Observer pattern where staff members are observers of plant state changes.

This pure virtual method is called when a plant needs attention. Derived classes must implement this method to define specific responses to plant care needs.
