Detailed Description

Holds API components for dependency injection.

Registers components in oatpp::base::Environment. Configures server, router, connection handlers and JSON mappers. All components created via Macros. For lower level implementation details visit the docs.

Oat++ flow: ConnectionProvider (TCP) -> Server -> HttpConnectionHandler -> HttpRouter
