Detailed Description

Concrete builder for constructing AverageCustomer objects with random dialogue selection.

Implements the CustomerBuilder interface to create AverageCustomer instances with randomly selected dialogues from the JSON configuration. Uses the Builder pattern to separate customer construction from customer representation.
